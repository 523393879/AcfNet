# AcfNet
This repository contains the code (in PyTorch) for "[Adaptive Unimodal Cost Volume Filtering for Deep Stereo Matching](https://arxiv.org/abs/1909.03751)".

## Requirements:
- PyTorch1.1+, Python3.5+, Cuda10.0+
